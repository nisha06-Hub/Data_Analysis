# Zomato Customer Review Analytics Dashboard using Microsoft Excel 
## Objective


To analyze customer reviews and transform raw feedback into actionable business insights using Microsoft Excel. This dashboard helps identify customer sentiment, common issue categories, recommended actions, and responsible departments through KPI cards, Pivot Tables, and Pivot Charts, enabling data-driven decision-making to improve customer satisfaction.

## Dataset
- <a href="https://github.com/nisha06-Hub/Data_Analysis/blob/main/Zomato_Data_Analysis.xlsx">Dataset</a>

## Business Questions Addressed

- What is the overall customer sentiment distribution?
- Which issue categories are most frequently reported?
- What are the most common recommended actions based on customer reviews?
- Which departments are responsible for handling customer concerns?
- Which areas require immediate business attention?
- How can customer feedback be used to improve customer satisfaction?
